# 📚 Book Review API

A RESTful API built with **Node.js**, **Express.js**, and **MongoDB** for managing books and reviews.  
Features include **JWT authentication**, **pagination**, **filters**, and **search**.

---

## 🚀 Features
- User authentication (JWT-based)
- Add, view, update, and delete reviews
- Add and fetch books (with filters + pagination)
- Search books by title/author (case-insensitive)
- Secure password hashing (bcrypt)
- Modular project structure

---

## ⚙️ Tech Stack
- Node.js + Express.js
- MongoDB (Atlas or local)
- JWT for authentication
- Mongoose ODM
- bcryptjs
